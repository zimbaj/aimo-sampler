# AIMO Sampler - Prompt Writing Guide

Master the art of writing prompts that generate exactly the sounds you want.

## The Basics

The AI understands natural language descriptions of sounds. The more specific and descriptive your prompt, the better results you'll get.

## Prompt Structure

### Basic Formula
```
[intensity/character] [sound type], [genre/style], [technical qualities], [extra details]
```

### Examples

**Kick Drums:**
```
punchy techno kick drum, tight transient, deep sub tail, electronic
hard industrial kick, distorted, aggressive attack, boomy
soft acoustic bass drum, warm, round, natural
```

**Snares:**
```
crispy electronic snare, bright, punchy, layered with clap
fat hip-hop snare, thick, compressed, powerful crack
acoustic snare with rimshot, bright, cutting, live drum sound
```

**Bass:**
```
deep psytrance bass, rolling, acid 303 style, distorted, aggressive
minimal techno sub bass, pure sine wave, clean, deep
reese bass, detuned saw waves, growling, drum and bass style
wobble bass, dubstep style, modulated, heavy, aggressive
```

**Synths:**
```
bright supersaw lead, trance style, detuned, powerful, epic
dark pad synthesizer, evolving, atmospheric, mysterious
pluck synth, short decay, bright attack, melodic, arpeggiated
```

**FX:**
```
cinematic impact hit, deep boom, reverberant, powerful
tension riser, building energy, increasing pitch, white noise
reverse cymbal swell, metallic, building, dramatic
```

## Keyword Reference

### Character/Intensity Words
| Word | Effect |
|------|--------|
| punchy | Strong transient, impactful |
| soft | Gentle, smooth |
| aggressive | Harsh, intense |
| warm | Pleasant, analog-like |
| cold | Digital, sterile |
| fat | Thick, heavy |
| thin | Light, narrow |
| crispy | High-frequency detail |
| muddy | Low-mid heavy |

### Texture Words
| Word | Effect |
|------|--------|
| clean | No distortion |
| distorted | Overdriven, saturated |
| gritty | Slightly distorted |
| smooth | Even, flowing |
| harsh | Aggressive high frequencies |
| saturated | Warm distortion |

### Space/Ambience Words
| Word | Effect |
|------|--------|
| dry | No reverb |
| reverberant | Lots of reverb |
| spacious | Wide, open |
| tight | Controlled, close |
| ambient | Atmospheric |
| intimate | Close, present |

### Technical Words
| Word | Effect |
|------|--------|
| transient | Attack portion of sound |
| sustain | Body of the sound |
| decay | Fade out |
| sub | Very low frequencies |
| high frequency | Treble content |
| modulated | Moving/changing |
| filtered | Frequency-shaped |

### Genre Words
| Genre | Associated Sounds |
|-------|-------------------|
| techno | minimal, driving, hypnotic, industrial |
| psytrance | rolling, acid, aggressive, psychedelic |
| house | groovy, four-on-floor, warm |
| dnb/drum and bass | fast, breakbeat, heavy bass |
| dubstep | wobble, heavy, aggressive, half-time |
| ambient | ethereal, spacious, evolving |
| cinematic | epic, dramatic, orchestral |

## Advanced Techniques

### Combining Descriptors
Layer multiple descriptors for more specific results:
```
dark aggressive psytrance bass, rolling, acid-like, heavily distorted, hypnotic
```

### Negative Space
Sometimes saying what it's NOT helps:
```
clean kick drum, no distortion, tight, not boomy, controlled sub
```

### Reference Styles
Reference well-known styles:
```
synthwave bass, 80s style, warm analog, detuned
berlin techno kick, industrial, powerful, warehouse sound
```

### Emotional Descriptors
The AI understands emotional context:
```
melancholic pad, sad, introspective, gentle, emotional
aggressive bass, angry, intense, powerful, driving
euphoric lead, uplifting, happy, bright, energetic
```

## Experimentation Tips

1. **Start simple, add detail:**
   - First: "kick drum"
   - Then: "techno kick drum"
   - Then: "punchy techno kick drum, tight, electronic"

2. **Generate variations:**
   ```powershell
   python -m src.cli generate "your prompt" --variations 5
   ```

3. **Use seeds to iterate:**
   ```powershell
   # Found a good one? Save the seed!
   python -m src.cli generate "bass" --seed 42
   ```

4. **Adjust temperature:**
   - Lower (0.7-0.9) for predictable results
   - Higher (1.1-1.3) for creative surprises

5. **Try genre mixing:**
   ```
   techno kick with acoustic character
   dubstep bass with psytrance rolling quality
   ```
